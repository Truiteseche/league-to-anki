# League to Anki

Generate Anki decks from the latest patch notes for League of Legends champions and items in any language
This project uses the official and up to date Riot Portal API

## Supported languages

Language support is defined by Riot portal API and includes these languages :

- English (ENG)
- French (FR)
- German (GER)
- Italian (IT)

## Customization options

- Edit the scheme of the cards
- Whitelist champions/items
- Include gameplay videos
- Include EnnemyTips and AllyTips
- ...
